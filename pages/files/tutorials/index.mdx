## How to Associate Files

ROQ's `createFileAssociation()` API allows files to be associated with multiple entities. This method will simplify the schema database because you don't need to change the existing schema database. Please read this tutorial on [how to associate file](/files/tutorials/how-to-associate-files) using this API.
